# 데이터 모델: 오늘의 할 일 웹 앱

## 엔티티

### TodoItem

할 일 항목을 나타내는 핵심 엔티티다.

- **id**: 할 일 식별자. 정수형이며 자동 증가한다.
- **title**: 할 일 제목. 문자열이며 비어 있을 수 없다.
- **completed**: 완료 여부. 기본값은 false다.
- **created_at**: 생성 시각. 날짜/시간 값으로 저장한다.
- **updated_at**: 마지막 수정 시각. 날짜/시간 값으로 저장한다.

### 제약 조건

- 제목은 빈 문자열 또는 공백만으로 구성될 수 없다.
- 완료 여부는 boolean 값이어야 한다.
- 생성 시각과 수정 시각은 서버에서 자동으로 설정한다.

## 상태 전이

- **미완료 → 완료**: PATCH 요청으로 completed=true를 전달해 변경한다.
- **완료 → 미완료**: PATCH 요청으로 completed=false를 전달해 변경한다.
- **생성**: POST 요청으로 새 할 일을 생성한다.
- **삭제**: DELETE 요청으로 특정 할 일을 제거한다.

## 조회 규칙

- `status=all`: 모든 할 일을 반환한다.
- `status=active`: 완료되지 않은 항목만 반환한다.
- `status=completed`: 완료된 항목만 반환한다.
